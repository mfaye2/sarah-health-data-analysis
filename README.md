# 📊 Analyse des données de santé - Sarah

## 🧠 Objectif
Ce projet vise à analyser les données de santé, de mode de vie et de dépenses de Sarah afin d’identifier des tendances et proposer des recommandations pour améliorer son bien-être global.

---

## 📂 Données utilisées

- 👟 Nombre de pas (activité physique)
- ❤️ Fréquence cardiaque
- 🔥 Calories brûlées
- 😴 Temps de sommeil
- 🤒 Symptômes (fatigue, maux de tête, nausée…)
- 😊 Humeur
- 🚴 Activité vélo
- 💰 Dépenses quotidiennes
- 🍽️ Habitudes alimentaires (photos des repas)

---

## 🛠️ Méthodologie

1. **Collecte des données**
   - Données issues de différentes sources (tracker, journal, dépenses)

2. **Nettoyage et consolidation**
   - Fusion des données dans un fichier Excel / Google Sheets
   - Standardisation des formats

3. **Analyse**
   - Analyse descriptive
   - Moyennes mobiles (ex : nombre de pas)
   - Corrélations (humeur, santé, dépenses)

4. **Visualisation**
   - Graphiques temporels (calories, activité…)
   - Tableaux croisés dynamiques

---

## 📊 Résultats clés

- 📉 Le nombre de pas diminue lorsque la charge de travail augmente
- 😴 Le sommeil est plus faible le week-end
- 🍔 Certains symptômes sont liés à l’alimentation
- 💸 Les dépenses augmentent lorsque l’humeur est basse

---

## 💡 Recommandations

- Augmenter l’activité physique en période de stress
- Améliorer le sommeil (surtout le week-end)
- Adapter l’alimentation pour réduire les symptômes
- Contrôler les dépenses en fonction de l’humeur

---

## 🧰 Outils utilisés

- Excel / Google Sheets
- Tableaux croisés dynamiques
- Graphiques
- Analyse de données
